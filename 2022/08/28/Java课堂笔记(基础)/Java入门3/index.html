java入门(第03章 运算符

第三章：运算符

算数运算符

类型

+，-，*，/，%，++，—

题型

关系运算符

  1. ==，!=，<，>，<=，>=
  2. 结果只有true和false

逻辑运算符

  1. &&：短路与，||：短路或
  2. &：逻辑与，|：逻辑或，^：逻辑异或
  3. !：取反

注意：逻辑 & 和 | 两边关系式都要判断（相当于通过两边关系式的返回值再运算）

赋值运算符

  • 基本赋值运算符：=
  • 复合赋值运算符：+=，-=，*=，、=，%=

三元运算符

  1. 条件表达式 ? 表达式1 : 表达式2;
  2. 条件为T，返回表达式1的结果；条件为F，返回表达式2的结果

二进制

  1. 二进制：以0b或0B开头
  2. 八进制：以0开头
  3. 十六进制：以0x或0X开头，不区分大小写
  4. 进制转换：略

位运算符

  • &，|，^，~，>>，<<，>>>
  • 算术左移和逻辑左移：都是整体向左移，在右边加0，若最高位不为1则相当与给该数乘以2的（位移数）次方。
  • 算术右移>>：将二进制整体向右移且左边加上符号位
  • 逻辑右移>>>：将二进制整体向右移且右边加上0

运算符优先级

标识符

定义：Java对各种 变量、变量名、方法、类、保留字等命名时使用的字符序列

  1. 变量由 0~9，_，$构成。数字不能开头、不能包含标识符
  2. 保留字：现有版本尚未使用，以后版本作为关键字使用（byValue、cast、future、generic、inner、operator、outer、rest、var、goto、const）

关键字
