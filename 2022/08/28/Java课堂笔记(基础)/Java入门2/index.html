第二章：变量

变量介绍

  • 变量是程序的基本组成单位
  • 变量三要素 (变量 + 名称 + 值)

+号的使用

  1. 当左右两边为数值型时，做加法运算

  2. 当左右两边有一方为字符串型时，做拼接运算

数据类型

所有数据类型

整形

类型 占用存储空间 范围
byte 1 字节 -127 ~ 128
short 2 字节 -32768 ~ 32767
int 4 字节 -2147483648 ~ 2147483647
long 8 字节 -2^63^ ~ 2^63^ -1

Java 的整型常量默认为 int 型，声明 long 型常量需加’l’或’L’，例：long a = 12L；，但 double 常量可以加 f

浮点类型

类型 占用储存空间 范围
单精度 float 4 字节
双精度 double 8 字节

注意：

  1. Java 的浮点型常量默认值为 double 型，声明 float 型需加’f’或’F’
  2. 比较 2.7 (直接赋值) 和 8.1 / 3 (计算出来的浮点数) 应该用差值的绝对值法判断 (java 中浮点数的赋值和运算都是用的近似值)
  3. 十进制表示方法：0.12，.12。科学计数法：5.12e2，5.12E2

字符类型

  • 范围：0 ~ 255
  • 可以是一个字母，数字，特殊符，汉字

布尔类型

不可以用 0 或非 0 的整数代替 true 和 false

编码

  1. Java 里面字符输出时，是 Unicode 编码对应的字符
  2. ASCII 相当于 Unicode 的一个子集
字符编码 所占空间大小 说明
ASCII 1 字节 可以表示 128 个字符
Unicode 字母和汉字都是 2 字节 包含世界上所有符号，浪费空间
utf-8 字母 1 字节，汉字 3 字节 在 utf-8 基础上进行改进，大小可变 (1-6 个字节表示一个符号，不同符号字节不同)
gbk 字母 1 字节，汉字 2 字节 可以表示汉字，范围广泛 (在控制台上就用这个)
gb2312 gb2312<gbk
big5 繁体中文，台湾，香港

Java 类的组成形式

数据类型转换

自动类型转换

介绍：当 Java 程序在进行赋值或运算时，精度小的类型自动转化为精度大的类型。

转换顺序：

  1. char - int - long - float - double
  2. byte - short - int - long - float - double

强制类型转换

介绍：自动转换的逆过程，将容量大的类型转换为容量小的，但可能造成精度降低或溢出

基本数据类型和 String 类型的转换

注意： 在 String 类型转换为 基本数据类型时，要确保能够转成有效的数据，如果格式不对，就会抛出异常，程序就会终止，例如不能把 hello 转换为 int 型
